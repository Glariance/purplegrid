## Instructions

Place your AI-related MP4 video files in this directory with the following names:

1. `ai-voice-cloning.mp4` - AI Voice Cloning Technology
2. `deep-learning-face-synthesis.mp4` - Deep Learning for Face Synthesis
3. `neural-networks-explained.mp4` - Neural Networks Explained
4. `machine-learning-basics.mp4` - Machine Learning Basics
5. `ai-content-generation.mp4` - AI Content Generation
6. `natural-language-processing.mp4` - Natural Language Processing
7. `computer-vision-image-ai.mp4` - Computer Vision & Image AI
8. `ai-video-processing.mp4` - AI Video Processing
9. `generative-ai-models.mp4` - Generative AI Models
10. `ai-automation-tools.mp4` - AI Automation Tools
11. `reinforcement-learning.mp4` - Reinforcement Learning
12. `ai-ethics-future.mp4` - AI Ethics & Future

## Video Requirements

- Format: MP4
- Recommended resolution: 1080p or higher
- Aspect ratio: 9:16 (vertical/portrait) for best display
- File size: Keep files optimized for web (under 50MB each recommended)
